## Run

- Clone Repo

    ```sh
    git clone https://github.com/lu-css/UberChallange
    ```

- Install Dependencies 

    ```sh
    dotnet restore
    ```


- Config `.env` file. Check the `.env.example` file.

- Start Server

    ```sh
    dotnet run
    ```

If in development environment, swagger will be started at `/swagger`.
